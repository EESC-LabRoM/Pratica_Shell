# The Shell

## Introdução

Nessa semana vocês tiveram a aula de Shell e tiveram os primeiros contatos com o terminal do Linux. Sabemos que nas primeiras vezes ele pode parecer assustador, mexer direto no terminal executar comandos de texto sem "ver" o que ta acontecendo. Nessa prática queremos desmitificar um pouco isso, ao longo do curso vocês irão aprender a ver e interpretar o terminal e no fim ele vai ser a sua principal interface com o computador.

## Atividades

### Explorador de Sistema

Nessa atividade o objetivo e aprender a explorar o sistema de vocês. Vocês devem fazer o seguinte:

- Abrir o terminal, descobrir qual diretório você está e listar todos os arquivos na pasta atual (inclusive os arquivos ocultos)

- Navegar até a `home` usando `cd` com **caminho absoluto** 

- Criar uma pasta chamada `pratica_shell` na pasta `Documents` do seu usuário. (Isso sem sair da pasta `home`)

- Andar pra pasta `pratica_shell` usando caminho relativo apartir da pasta atual

### Construindo um projeto

- Dentro da pasta `pratica_shell` deve criar a seguinte estrutura de pasta: `pratica_shell/docs`, `pratica_shell/src` e `pratica_shell/data`

- Criar o arquivo `hello_world.cpp` na pasta `src`

- Criar o arquivo `README.md` na `pratica_shell/` e escrever seu nome, numero USP e o nome do seu lider responsável utilizando `nano`.

- Criar o arquivo `docs.md` na pasta `pratica_shell/docs` e escrever um nome de um professor seu (da USP).

- Além disso, no repositorio, vocês tem acesso aos seguintes arquivos: `dados.csv` e `logs.txt`. Movam `dados.csv` utilizando linha de comando para `pratica_shell/data`

- Também movam `logs.txt` para `pratica_shell/data` via linha de comando. 

### Inspecionando um arquivo

- Veja quantas linhas tem o arquivo de log em `pratica_shell/data/logs.txt`

- Analise as 5 primeiras linhas e também as 5 ultimas linhas

- Veja se tem e em qual linha se encontra um erro. (Dica: use `grep`)

### Copiando

Faça usando linha de comando:

- Copie o arquivo de log e passe para um outra pasta `pratica_shell/logs/`

- Mova o arquivo de dados para `pratica_shell/recordings/`
